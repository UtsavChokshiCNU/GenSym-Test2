Configuring Level 2 Parameters for a Manufacturer Role

To configure the Level 2 parameters of a Manufacturer role, it is helpful to think about the manufacturing process in terms of the upstream source, the make, and the downstream delivery subprocesses. Each of these subprocesses has parameters you can configure.

The following list of configurable parameters assumes a stock/replenishment planning strategy, a single delivery product, a single supplier for each source product, and a pull planning mode. See the related topics for information about additional parameters you must configure for each of these alternative modeling techniques.

Note: The ES: Enable Source and ED: Enable Delivery categories have no configurable parameters.
Upstream source subprocess
Source category:
- Receive and Putaway Time
- Product Acquisition, Create Customer Order, and Supply Chain Finance costs
P2: Plan Source category:
- Plan Type
- Planning Period
- Initial Plan Delay
- Replenishment Planning
Source products:
- Customer Preference
- Starting Inventory
- Inventory Maximum
- Desired Turnaround
- Order Fulfillment Preference
- Receive and Putaway Time multiplier
- Replenishment Threshold
- Minimum Reorder Quantity
- Maximum Threshold
Make subprocess
Make categories:
- Batch Preference
- Build Selection parameters of M1: Make-to-Stock category
- Order Release to Build Time
- Production Material Cycle Time
- Total Build Time
- Move to Delivery
- Production Material Handling cost
- ECO and Engineering Time of the M3: Engineer-to-Order category
P3: Plan Make category:
- Plan Type
- Planning Period
- Initial Plan Delay
- Replenishment Planning
- Compensate for Yield
Delivery products:
- Maximum Batch Size
- Minimum Batch Size
- Yield
- Build Time Dependent on Order Size
- Total Build Time, Order Release to Build Time, Production Material Cycle Time, and Move to Delivery multipliers
- Engineering Time multiplier of engineer-to-order products
Downstream delivery subprocess
Deliver categories:
- Order Receipt to Entry Time
- Order Selection parameters of D1: Deliver Stocked Product category
- Pick Material Time
- Packing Time
- Order Shipment to Receipt Time
- Carrier
- Order Entry, Order Fulfillment, Distribution, Transportation, Customer Invoicing, and Customer Collections costs
Delivery products:
- Product Preference
- Starting Inventory
- Inventory Maximum
- Completion Preference
- Net Selling Price
- Published Delivery Lead Time
- Order Receipt to Entry Time, Pick Material Time, and Packing Time multipliers
- Replenishment Threshold
- Minimum Reorder Quantity
- Maximum Threshold
