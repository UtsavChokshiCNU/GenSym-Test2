NXP TriMedia Compilation System

Build Run-Time Library

Use the commands below to build the BullseyeCoverage run-time library. Modify the compile command to use the same compile options as your project. 

cd %ProgramFiles%\BullseyeCoverage\run
tmcc -c -O libcov-trimedia.c
tmar -r ../lib/libcov-trimedia.a libcov-trimedia.o

Modify NXP Makefiles

In the directory c:/Nexperia/NDK5.6/sde/tm_psos, modify common.mk to change the setting of CC to tmcc: 

CC = tmcc

Modify maketarget_dvp1.mk to add the BullseyeCoverage run-time library to the link command: 

LOCAL_LDFLAGS += c:/Progra~1/BullseyeCoverage/lib/libcov-trimedia.a \

Build Project

Add the BullseyeCoverage/bin directory to the front of the PATH environment variable. Enable coverage build and then build as you normally do. 

set PATH=%ProgramFiles%\BullseyeCoverage\bin;%PATH%
cov01 -1
(rebuild project)
