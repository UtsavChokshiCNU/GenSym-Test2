Quick Start on Windows

  1. Check for special instructions for your compiler and platform in Building with Specific Tools and Platforms.
  2. Enable Coverage Build. Do this with the Coverage Browser menu command Tools Coverage Build. In the Microsoft Visual Studio, you can use menu command Tools Enable/Disable BullseyeCoverage Coverage Build.
  3. Rebuild and run your test program as you normally do.
  4. Run the Coverage Browser to view results.
