Error LNK2001 Unresolved Symbol on DLL Entry Point

SYMPTOMS

The errors shown below occur when linking a DLL. 

BullseyeCoverageEntryUser-n.obj : error LNK2001: unresolved external symbol DllMain@12
fatal error LNK1120: 1 unresolved externals

BullseyeCoverageEntryUser-n.obj : error LNK2001: unresolved external symbol "int __stdcall DllMain(struct HINSTANCE__ *,unsigned long,void *)"
fatal error LNK1120: 1 unresolved externals

CAUSE

Your DLL entry point DllMain parameters and return type do not conform to the Win32 DllMain Callback Function: 

BOOL WINAPI DllMain(HINSTANCE hinstDLL, DWORD fdwReason, LPVOID lpvReserved);

RESOLUTION

Modify your source code to change your DllMain callback function so that the parameters and return type match the Win32 documentation.
