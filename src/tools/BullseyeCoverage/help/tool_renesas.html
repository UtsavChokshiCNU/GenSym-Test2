Renesas High-performance Embedded Workshop

The Renesas High-performance Embedded Workshop cannot be configured to build using BullseyeCoverage. However, you can build from the command prompt using a makefile as described below. This documentation applies to HEW version 4.7.

Port The Run-Time Library

For detailed information on porting the BullseyeCoverage run-time library, see Custom and Embedded Environments

Build with a Makefile

  1. Use the menu command Build Generate Makefile. Load the resulting makefile into a text editor.

  2. Add the BullseyeCoverage\bin directory at the front of the PATH setting. For example, insert the underlined text below. 
    set PATH=c:\Program Files\BullseyeCoverage\bin;$(TC_INSTALL_DIR)\bin

  3. Change all invocations of the compiler from absolute to relative. For example, remove the lined-through text below. 
    "$(PROJECT_SIMDEBUG_SH1_SH_CC_LIBRARY_GENERATOR01_EXEDIR)\$(PROJECT_SIMDEBUG_SH1_SH_CC_COMPILER02_EXE)" -subcommand=<<

  4. Perform a clean build using the Renesas make program. For example: 
    del ..\..\workspace\project\Debug\*.obj
cov01 -1
hmake -f project.mak
