AIX Kernel

  1. Add BullseyeCoverage/lib/libcov-kernel.a to your link command.
  2. Export the symbol cov_kernel. For example, add the ld option -Bexport=cov_kernel to your link command.
  3. Enable coverage build and build as usual.
  4. After testing your kernel code, run the command covgetkernel to save coverage measurements from memory to the coverage file.
  5. View a coverage report as usual.

Environment variable COVAUTOSAVE has no effect.
