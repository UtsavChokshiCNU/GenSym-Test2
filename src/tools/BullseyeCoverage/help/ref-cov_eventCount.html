cov_eventCount - Count Coverage Events 

#include <BullseyeCoverage.h>
unsigned cov_eventCount(void);

DESCRIPTION

Function cov_eventCount returns the total number of coverage events recorded in memory for the current executable image.

RETURN VALUE

cov_check returns the sum of the following.

  • the number of functions invoked
  • the number of conditions and decisions evaluated true
  • the number of conditions and decisions evaluated false
  • the number of case and default labels encountered
  • the number of C++ catch handlers encounter

EXAMPLES

#if _BullseyeCoverage
	const unsigned count = cov_eventCount();
#endif

testCase1();

#if _BullseyeCoverage
	if (count == cov_eventCount())
		printf("testCase1 does not increase coverage\n");
#endif
