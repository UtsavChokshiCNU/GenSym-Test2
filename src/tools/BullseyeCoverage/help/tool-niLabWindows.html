National Instruments LabWindows/CVI

You can use BullseyeCoverage with LabWindows/CVI projects built with an external compiler, such as Microsoft C++. This information applies to LabWindows/CVI 9.1.

Configure Project

  1. Set up the project to build with an external compiler. For detailed instructions, see the LabWindows/CVI help topic Creating Optimized Code.
  2. Set the Release configuration with the LabWindows/CVI command Build Configuration Release
  3. Verify a successful build with the LabWindows/CVI command Build Create Release Executable
  4. Add the BullseyeCoverage 32-bit Windows run-time library to the project. To do this, use the LabWindows/CVI command Edit Add Files to Project Library and select the file BullseyeCoverage\lib\libcov-windows-x86.lib.

Build and Run

  1. Enable coverage build
  2. Rebuild with the LabWindows/CVI commands Build Mark Project for Compilation, then Build Create Release Executable
  3. Run as usual.
