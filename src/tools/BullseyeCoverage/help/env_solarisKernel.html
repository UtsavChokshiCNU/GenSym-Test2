Solaris Kernel

BullseyeCoverage supports measuring coverage of Solaris SPARC kernel modules. However, the module exit function _fini cannot be measured.

  1. Enable coverage build and build as usual.
  2. Link each of your modules with the BullseyeCoverage library libcov-kernel64.a (for the 64-bit kernel) or libcov-kernel32.a (for the 32-bit kernel).
  3. After testing your modules, run the command covgetkernel64 (for the 64-bit kernel) or covgetkernel32 (for the 32-bit kernel) to save coverage measurements from memory to the coverage file. To run covgetkernel64 on Solaris 8, you may need to apply patch 108435. To run covgetkernel64 on Solaris 7, you may need to apply patch 106300.
  4. View a coverage report as usual.

Environment variable COVAUTOSAVE has no effect.
