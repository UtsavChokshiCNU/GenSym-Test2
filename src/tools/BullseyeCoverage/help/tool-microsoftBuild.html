Building with Microsoft BUILD

Before using the Microsoft Build Engine build.exe, first use set PATH= to put the BullseyeCoverage bin directory first in the PATH environment variable. Use cov01 -1 to enable Coverage Build; do not make changes to any files. For example, 

set PATH=%ProgramFiles%\BullseyeCoverage\bin;%PATH%
cov01 -1
build -c
