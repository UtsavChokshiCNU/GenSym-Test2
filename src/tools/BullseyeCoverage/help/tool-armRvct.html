ARM RealView Compilation Tools

BullseyeCoverage supports ARM Developer Suite 1.2 through ARM RealView Compilation Tools Version 3.1.

Build Run-Time Library

Use the commands below to build the BullseyeCoverage run-time library. Modify the compile command to use the same compile options as your project. 

cd BullseyeCoverage/run
armcc -c libcov-armRvct.c
armar --create ../lib/libcov-armRvct.a libcov-armRvct.o

Build Project

Make sure the BullseyeCoverage/bin is first in the PATH environment variable. Enable Coverage Build. Then launch the IDE from the shell where you modified PATH. Rebuild your application. 

set PATH=%ProgramFiles%\BullseyeCoverage\bin;%PATH%
cov01 -1
(launch IDE/rebuild)

Saving Coverage

If your application never exits, add a call to cov_write to your source code, at a point you know to be executed after the code you want to measure. 

#if _BullseyeCoverage
	cov_write();
#endif

Run Semi-Hosted

Run your application in a semihosted environment. BullseyeCoverage run-time error messages are written to standard error.
