Sun C++ Undefined Symbol in SunWS_cache

SYMPTOMS

Undefined          first referenced
 symbol                in file
 cov_o_data_...         .../SunWS_cache/....o
 cov_o_...              .../SunWS_cache/....o
 ld: fatal: Symbol referencing errors. No output written

CAUSE

You built more than one program using the same template repository. This is not allowed by the Sun C++ compiler.

RESOLUTION

Remove the template repository before building each program, with the command below. 

CCadmin -clean

MORE INFORMATION

This restriction is described in the Sun C++ User's Guide 6.7.7 Building Multiple Programs Using Templates in the Same Directory
