Visual C++ 6 "cov113: cannot run program cl.exe: Invalid argument"

SYMPTOMS

With Microsoft Visual C++ 6 and with coverage Build enabled, this error occurs: 

error cov113: cannot run program ...\cl.exe: Invalid argument

RESOLUTION

Install Visual C++ 6 service pack 5.
