Eclipse

You can use BullseyeCoverage with the Eclipse Platform and the C/C++ Development Tooling. This documentation applies to Eclipse 3.4, 3.3 and 3.2.

Eclipse 3.3, 3.4

To configure Eclipse, follow these steps.

  1. Use the Eclipse menu command Project Properties.
  2. Select C/C++ Build: Settings
  3. Select the C++ compiler, for example GNU C++ Compiler
  4. Change the Command setting to the full path of the BullseyeCoverage C++ interceptor. For example, change g++ to "c:/Program Files/BullseyeCoverage/bin/g++" (on Windows), or /opt/BullseyeCoverage/bin/g++ (on Unix).
  5. Perform steps 3-4 for the C compiler and the linker

Eclipse 3.2 Windows Managed Make Project

To configure Eclipse, follow these steps.

  1. Use the Eclipse menu command Project Properties.
  2. Select C/C++ Build
  3. Select Environment
  4. Click New
  5. Enter the name PATH
  6. For the value, enter BullseyeCoverage\bin followed by the existing setting for PATH, shown the System Variables section.

Eclipse 3.2 Windows Standard Make Project

Make sure BullseyeCoverage/bin is at the front of the PATH environment variable in the context you launch Eclipse.

Unix

Make sure BullseyeCoverage/bin is at the front of the PATH environment variable in the context you launch Eclipse. For example: 

PATH=$HOME/BullseyeCoverage/bin:$PATH eclipse
