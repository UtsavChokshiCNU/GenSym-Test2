LINK: error: Internal error during Pass1

SYMPTOMS

When building using Visual C++ 5.0 you see the error below. 

LINK: error: Internal error during Pass1

CAUSE

The problem occurs with Microsoft link.exe version 5.00.7022.

RESOLUTION

Install Visual Studio 97 Service Pack 3 which includes Microsoft link.exe version 5.10.7303.
