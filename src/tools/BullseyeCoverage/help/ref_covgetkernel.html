covgetkernel - Save Unix Kernel Measurements 

covgetkernel [options]

DESCRIPTION

The covgetkernel command retrieves coverage measurements collected by instrumentation inside the operating system kernel and saves them to the coverage file named by environment variable COVFILE. You must be the super-user to run this command.

To accommodate for variations in hardware such as 32-bit versus 64-bit, we provide more than one command named covgetkernelsuffix, where the suffix might 32 or 64, for example.

On Linux x86 and Linux x64, the directory BullseyeCoverage/bin-embeddedLinuxKernel/ contains versions of this program for popular embedded Linux architectures.

OPTIONS

-d--debug Output diagnostic information for technical support.
-fPath--file Path The coverage file is specified by Path. The default is test.cov.
-r--reset Reset the kernel measurements in memory to zero. After this user mode program resets the measurements, the action of the kernel unloading the covgetkernel program might immediately causes some measurements. This option is provided only on AIX, Linux, HP-UX, Solaris, and Tru64.
