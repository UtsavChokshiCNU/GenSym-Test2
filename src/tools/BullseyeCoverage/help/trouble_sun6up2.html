Sun C++ "End of file while skipping text"

SYMPTOMS

With Sun Forte C++ 6 Update 2 and with Coverage Build enabled, the warning below occurs. 

".../definitions", line 1: Warning (Anachronism): End of file while skipping text -- Check #if .. #endif pairing.

Apart from the warning, the build completes successfully.

CAUSE

This is due to a bug in the Sun Forte C++ compiler.

RESOLUTION

On SPARC systems, apply Sun patches 111678 and 111685. You can obtain Sun patches from sunsolve.sun.com.
