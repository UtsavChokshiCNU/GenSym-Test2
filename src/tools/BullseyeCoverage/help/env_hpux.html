HP-UX

BullseyeCoverage supports HP-UX 11.0 and later.

PARISC 32-bit Shared Objects

The patches listed below are required for measuring coverage of PARISC 32-bit shared objects.

OS revisionRequired patches
11.23 PHSS_35978 s700_800 11.23 aC++ Runtime
11.11 PHSS_33945 s700_800 11.11 HP aC++ -AA runtime libraries
PHSS_35381 s700_800 11.11 ld(1) and linker tools cumulative patch
11.0 PHSS_33943 s700_800 11.x HP aC++ -AA runtime libraries
PHSS_35380 s700_800 11.00 ld(1) and linker tools cumulative patch

Kernel

BullseyeCoverage supports measuring coverage of the HP-UX kernel.

  1. Enable coverage build and build as usual.
  2. Link with one of the libraries listed below.
    • libcov-hpKernel-ia64.a.
    • libcov-hpKernel-pa64.a.
  3. After testing your kernel code, run the command listed below to save coverage measurements from memory to the coverage file. Use the command corresponding to the libraries listed above.
    • covgetkernel-ia64
    • covgetkernel-pa64
  4. View a coverage report as usual.

Environment variable COVAUTOSAVE has no effect.
